# DDE File Manager External

A collection of external plugins for Deepin/UOS File Manager.

## Dependencies

- Qt >= 5.11
- CMake >= 3.13
- DtkCore
- DtkWidget
- DtkGui

## Build and Install

### Build from source

```bash
mkdir build
cd build
cmake ..
make
```

### Install

```bash
sudo make install
```
